# Everything

## 一句话介绍
Windows 平台上极快的本地文件搜索工具，适合替代系统自带搜索。

## 链接
- 官网：https://www.voidtools.com/

## 适合谁
- 文件很多的人
- 经常要找文档、代码、图片的人

## 为什么值得推荐
- 搜索速度非常快
- 结果直给，不绕弯子
- 很适合做系统级常驻工具

## 我最喜欢的地方
- 基本会让你怀疑为什么系统自带搜索能慢成那样

## 不足或注意点
- 功能上偏专注，不是全能型工作流工具
- 主要是 Windows 体验最好

## 类似选择
- Listary
- Fluent Search
