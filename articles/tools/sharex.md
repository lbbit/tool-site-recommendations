# ShareX

## 一句话介绍
Windows 上非常强大的截图、录屏、OCR 和自动化分享工具。

## 链接
- 官网：https://getsharex.com/
- GitHub：https://github.com/ShareX/ShareX

## 适合谁
- 经常截图、录屏、做教程的人
- 想把上传、标注、复制链接自动化的人

## 为什么值得推荐
- 功能非常全
- 自动化能力强
- 免费开源且长期维护

## 我最喜欢的地方
- 很多原本要来回切软件的动作，它一套就能做完

## 不足或注意点
- 新用户第一次打开会觉得选项有点多
- 主要面向 Windows

## 类似选择
- Snipaste
- Flameshot
