# PicGo

## 一句话介绍
一个帮助你快速上传图片到图床并返回链接的桌面工具。

## 链接
- GitHub：https://github.com/Molunerfinn/PicGo

## 适合谁
- 经常写 Markdown 的人
- 需要把本地图片快速转成外链的人

## 为什么值得推荐
- 对写博客和记笔记很省事
- 支持多种图床方案
- 拖拽上传体验直接

## 我最喜欢的地方
- 配好之后，插图这件事会顺滑很多

## 不足或注意点
- 图床配置本身可能有门槛
- 长期稳定性取决于你自己的图床选择

## 类似选择
- uPic
- ImageBed
